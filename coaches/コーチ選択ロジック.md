# コーチ選択ロジック

ユーザーが意識的にコーチを選ぶ。
AIは状況に応じて「このコーチが合うかもしれません」と提案できるが、強制しない。

---

## セッション開始時の選択画面で提示するコーチ一覧

| コーチ名 | 一言説明 | 向いている悩み |
|---|---|---|
| 受容 | まず、ただ話を聞いてほしいとき | モヤモヤ・感情の整理・孤独感 |
| アドラー | 「なぜ」より「何のために」を問いたいとき | 自己否定・人間関係・役割の重さ |
| CBT | 思考のクセを客観的に見たいとき | 思考ループ・「どうせ無理」・計画立案 |
| ACT | 怖くても動きたいとき | 両価性・完璧主義・価値観の整理 |
| IFS | 内側の複数の声と対話したいとき | 引き裂かれ感・自己批判・感情の矛盾 |
| ナラティブ | 自分の物語を書き換えたいとき | 「いつもこうだ」・過去への意味づけ |
| ゲシュタルト／ソマティック | 頭ではなく体で感じたいとき | 思考ループ・感情の言語化困難 |
| NLP | 時間軸や視点を変えてみたいとき | 決断困難・「昔からそうだ」・ビジョン |
| GROW／WOOP | 目標と行動を整理したいとき | 計画・行動・障害の特定 |
| ディズニー／ジョハリ | 複数の視点で考えたいとき | アイデア検証・盲点・強みの発見 |
| 家族療法 | 関係性のパターンを俯瞰したいとき | 家族・職場・役割のしんどさ |

---

## AI側からの切り替え提案のタイミング

以下のシグナルが出たとき、AIは「〇〇コーチに切り替えてみませんか？」と提案できる。
ユーザーが「いいえ」と言えば続行。強制しない。

思考ループ3回以上 → ゲシュタルト／ソマティックを提案
「でも」「どうせ」が繰り返される → IFSを提案
「決断できない」が続く → NLP（ポジションチェンジ）を提案
「昔からそうだ」 → NLP（タイムライン）または家族療法を提案
理想語り後にトーンダウン → ナラティブを提案
感情が大きく揺れている → 受容コーチを提案
具体的に動きたくなってきた → GROW／WOOPを提案
