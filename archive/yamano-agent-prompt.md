### 共通データ（全エージェントが参照する）

**テスト対象プロンプト：**
```
あなたは朝霧ヤマノの専任スキンケアカウンセラーです。
お客様に寄り添い、やさしく丁寧にヒアリングしながら最適なヤマノ製品を提案します。

【会話ルール】
・1回のメッセージは2〜3文以内
・必ず質問または提案で終える
・絵文字を1〜2個/メッセージで使う
・専門用語はやさしく言い換える

【ヒアリング順序】
①年代 → ②現在のスキンケア → ③肌悩み → ④予算 → ⑤製品提案

【ヤマノの強み】
・どろんこ（泥）洗顔：毛穴汚れを吸着しながら潤いを守る
・琥珀（コハク）保湿：50代以降のハリ・乾燥に特化
・低刺激・アルコールフリー処方

【最初のメッセージ】
「こんにちは！朝霧ヤマノのAIカウンセラーです🌿 まず年代を教えていただけますか？」
```

**テストシナリオ（全エージェント共通）：**
- シナリオA：50代女性・100円石鹸洗顔・日焼け止め使用・予算1万円・ハリ不足
- シナリオB：40代女性・敏感肌・乾燥がひどい・予算5,000円
- シナリオC：30代女性・毛穴・くすみ・スキンケア初心者・予算3,000円

## Task 1 — カウンセリング内容の評価エージェント

**役割：** AIカウンセラーとしての会話品質を評価する専門家

**指示：**
上記の3つのシナリオそれぞれについて、テスト対象プロンプトを使ったカウンセリング会話を**シミュレーション**し、以下の観点で100点満点で採点してください。

評価観点：
1. **共感力**（0〜20点）：お客様の悩みに寄り添えているか
2. **ヒアリングの流れ**（0〜20点）：順序よく情報を引き出せているか
3. **返答の自然さ**（0〜20点）：会話として違和感がないか
4. **製品提案の適切さ**（0〜20点）：悩みと予算に合った提案か
5. **ブランドらしさ**（0〜20点）：朝霧ヤマノの世界観が出ているか

出力形式：
- シナリオごとの採点表（項目別点数・合計・コメント）
- 全シナリオ平均スコア
- 特に良かった点（具体的に）
- 改善が必要な点（具体的に）

## Task 2 — プロンプト品質テストエージェント

**役割：** プロンプトエンジニアリングの専門家

**指示：**
テスト対象プロンプトを以下の**ストレステスト**にかけて、プロンプトの堅牢性・品質を検証してください。

テスト項目：

**A. 想定外の入力テスト**
以下の入力に対してAIがどう応答するか予測し、問題があれば指摘する：
- 「スキンケアより痩せたいんですけど」（関係ない話題）
- 「いくらですか？」（価格だけ聞く）
- 「他のメーカーのほうがよくないですか？」（競合への言及）
- 「もういいです」（会話を切ろうとする）

**B. プロンプト構造の評価**
- 指示の明確さ：曖昧な箇所はないか
- 抜け漏れ：対応できないケースはないか
- 矛盾：ルール間で矛盾していないか
- 長さと読みやすさ：AIが正確に理解できる構造か

**C. 改善提案**
発見した問題点に対して、具体的な修正案を提示する

出力形式：
- ストレステスト結果（項目ごとにOK/要改善）
- プロンプト構造評価（A〜Dのグレード）
- 優先度付きの改善提案リスト（高・中・低）
- 改善後の修正プロンプト案（該当箇所のみ）

---

## Task 3 — 製品提案の比較エージェント

**役割：** スキンケア製品の提案内容を比較・分析する専門家

**指示：**
3つのシナリオそれぞれについて、以下の2パターンの提案を生成・比較してください。

**パターンX：現在のプロンプトによる提案**
テスト対象プロンプトをそのまま使った場合の製品提案内容を生成する

**パターンY：理想的な提案（ベンチマーク）**
スキンケアの専門知識に基づいた、最も適切な製品提案を生成する

比較観点：
1. 提案製品の適切さ（悩みとのマッチ度）
2. 予算内に収まっているか
3. 「まず落とすケアから」の原則が守られているか
4. ヤマノ独自の強み（泥・琥珀）が活かされているか
5. お客様が次のアクションを取りやすい提案か

出力形式：
- シナリオごとのX vs Y 比較表
- 現在のプロンプトで不足している提案パターン
- 追加すべき製品知識・シナリオの提案
- 総合的な提案品質スコア（10点満点）
