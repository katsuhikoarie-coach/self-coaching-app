## 変更点サマリー（Before / After）

### 変更1｜スコープ・応答ポリシーの追加（高優先度）
| | 内容 |
|--|------|
| **Before** | スコープ外・競合・価格・離脱に対するルールがゼロ。架空価格のハルシネーションや競合言及のリスクあり。「必ず質問で終える」ルールが離脱時に矛盾。 |
| **After** | 4シナリオ別の返答テンプレートを明記。①スコープ外→お肌の話題へ誘導、②競合→言及禁止ポリシー、③価格→公式サイト案内、④離脱シグナル検知→質問ルール例外として完結フロー。 |

### 変更2｜製品提案の論理構造化（高優先度）
| | 内容 |
|--|------|
| **Before** | 製品名を列挙するだけ。「なぜ変える必要があるか」の文脈なし。予算に対し詰め込みすぎる傾向。継続イメージも提示なし。 |
| **After** | 3ステップ構造（STEP1：問題提起→STEP2：予算連動の提案数制限→STEP3：今日・1ヶ月・3ヶ月のロードマップ）を義務化。予算3,000円→1点、5,000円→2点、10,000円→3点の制限ルールを明記。 |

---

### 変更3｜年代別・悩み別 製品マッピングの追加（高優先度）
| | 内容 |
|--|------|
| **Before** | どの年代・悩みにどの製品を提案するかの判断基準なし。担当者によってバラつきが発生するリスク。 |
| **After** | 30代（毛穴・皮脂）／40代（バリア・揺らぎ）／50代（エイジング）の優先訴求軸と、悩み×年代→推奨製品のマッピングテーブルを追加。 |

---

### 変更4｜クロージング・次のアクション提示の追加（中優先度）
| | 内容 |
|--|------|
| **Before** | 提案後「ご興味はありますか？」止まり。初心者への使い方説明なし。購買への動線がない。 |
| **After** | 提案後に必ず「次のアクション1つ」を提示するルールを追加。初心者向けに使用量・手順・頻度を1文で添えるルール（どろんこ洗顔・琥珀保湿の例文付き）を明記。 |

---

### 変更5｜ブランド言語ガイドラインの追加（中優先度）
| | 内容 |
|--|------|
| **Before** | 汎用的な表現が多く朝霧ヤマノらしさが薄い。「山の恵み」「朝の霧」などの世界観ワードが未活用。最初のメッセージも無個性。 |
| **After** | 「山の恵み」「朝の澄んだ空気」「清らかな成分」など5例の推奨表現を定義。1メッセージ1表現まで・自然に組み込む注意点も付記。最初のメッセージも世界観ワードを盛り込んだ文言に更新。 |
