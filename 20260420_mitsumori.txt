```sql
CREATE TABLE IF NOT EXISTS quotes (
  id          INT AUTO_INCREMENT PRIMARY KEY,
  quote_no    VARCHAR(20) NOT NULL UNIQUE,
  cust_no     VARCHAR(10) NOT NULL DEFAULT '',
  cust_name   VARCHAR(100) NOT NULL DEFAULT '',
  quote_date  DATE NOT NULL,
  rate        INT NOT NULL DEFAULT 100,
  note        TEXT,
  total_tax10 INT NOT NULL DEFAULT 0,
  total_tax8  INT NOT NULL DEFAULT 0,
  total_amount INT NOT NULL DEFAULT 0,
  created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at  DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

CREATE TABLE IF NOT EXISTS quotes_items (
  id         INT AUTO_INCREMENT PRIMARY KEY,
  quote_id   INT NOT NULL,
  sort_order INT NOT NULL DEFAULT 0,
  prod_code  VARCHAR(20) NOT NULL DEFAULT '',
  prod_name  VARCHAR(200) NOT NULL DEFAULT '',
  price      INT NOT NULL DEFAULT 0,
  qty        INT NOT NULL DEFAULT 1,
  rate       INT NOT NULL DEFAULT 100,
  tax_rate   INT NOT NULL DEFAULT 10,
  FOREIGN KEY (quote_id) REFERENCES quotes(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

**内容：**

`api/sales.php` を参考にしながら、以下の仕様で作成してください。

**GET：見積一覧または1件取得**
- `?id=XX` → 該当見積＋明細を返す
- パラメータなし → 見積一覧（quotes）を新しい順で返す

**POST action=create：新規見積保存**
- quotes に1件INSERT
- quotes_items に明細を複数INSERT
- quote_no は `Q` + 年月日 + 連番4桁（例：Q20260420-0001）で自動採番

**POST action=update：見積修正**
- quotes を UPDATE
- quotes_items は一旦DELETE → 再INSERT

**POST action=delete：見積削除**
- quotes を DELETE（CASCADE で quotes_items も自動削除）

**レスポンス形式（一覧）：**
```json
[
  {
    "id": 1,
    "quote_no": "Q20260420-0001",
    "cust_no": "005",
    "cust_name": "有江千種",
    "quote_date": "2026-04-20",
    "rate": 80,
    "note": "",
    "total_amount": 12000,
    "items": []
  }
]
```

**レスポンス形式（1件）：**
items配列に明細を含めて返す

**認証：**
ファイル冒頭に必ず以下を記述
```php
require_once __DIR__ . '/../config/db.php';
```
